2. 수집하는 개인정보

본 서비스는 별도의 개인정보를 수집하지 않습니다.

회원가입, 로그인, 이름·이메일·전화번호 등 어떠한 개인식별정보도 요구하지 않습니다.

3. 기기 내 저장 데이터

서비스 이용 중 아래 정보가 사용자의 기기(브라우저 localStorage)에만 저장됩니다.

  • 학습 진도 (완료한 한자, 급수)
  • 복습 일정 (간격 반복 학습 데이터)
  • 쓰기 연습 점수 기록
이 데이터는 외부 서버로 전송되지 않으며, 브라우저 설정에서 언제든지 삭제할 수 있습니다.

4. 제3자 제공

수집된 개인정보가 없으므로 제3자에게 제공하는 정보가 없습니다.

5. 사용하는 브라우저 기능

  • Web Speech API (TTS): 한자 훈음을 음성으로 읽어주는 기능. 음성 합성은 기기 내에서 처리되며 외부로 전송되지 않습니다.
  • Web Audio API: 효과음 재생. 외부 서버 연동 없음.
  • localStorage: 학습 데이터 기기 내 저장.
  • Service Worker: 오프라인 사용을 위한 파일 캐싱.

6. 한자 데이터 출처

제공되는 한자 획순 데이터는 공개된 교육용 데이터를 기반으로 구성되었습니다.
